# FinVibe 📈
A modern full-stack stock trading and investment platform inspired by Zerodha. Built using a monorepo architecture with dedicated services for users, admins, and data processing.

## 📁 Project Structure
- **/frontend**: The user-facing web application for trading and portfolio tracking.
- **/backend**: Secure Node.js/Express REST API handling authentication, orders, and data.
- **/dashboard**: Admin and analytics dashboard for advanced market insights.

## 🛠️ Tech Stack
- **Frontend & Dashboard:** React.js, Vite, Axios, Chart.js / Recharts
- **Backend:** Node.js, Express.js, JWT (JsonWebToken)
- **Database:** MongoDB (via Mongoose)

## 🚀 Deployment Info
This project is structured as a monorepo and is ready for deployment on **Render** using separate root directories for each service.
